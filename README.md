## ✨ Features

### 🛡️ Authentication
- Register/Login with JWT
- Role-based access (Student/Instructor/Admin)
- Protected routes
- Profile management

### 📚 Course Management
- Create, Read, Update, Delete courses
- Publish/Unpublish courses
- Course search & filters
- Course categories
- Course enrollment
- Progress tracking

### 🤖 AI Integration
- AI course generator
- AI tutor chat
- AI quiz generator
- AI content summarizer
- AI concept explainer

### 💬 Real-time Chat
- Instant messaging
- Online/offline status
- Typing indicator
- Read receipts
- Message notifications

### ⭐ Reviews & Ratings
- Rate courses (1-5 stars)
- Write reviews
- Edit/Delete reviews
- Average rating display
- Verified purchase badge
- Instructor reply

### 👨‍💼 Admin Dashboard
- Platform statistics
- User management
- Course moderation
- Revenue analytics
- Charts & reports

### 🔔 Notifications
- In-app notifications
- Email notifications
- Real-time alerts
- Notification preferences

### 🎓 Advanced Features
- Wishlist/Favorites
- Certificate generation
- Student analytics
- Instructor analytics

## 🛠️ Tech Stack

### Frontend
- React 18 + Vite
- React Router v6
- Framer Motion
- Recharts
- Socket.io Client
- React Hot Toast

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Socket.io
- Nodemailer
- Google Gemini AI
- Cloudinary

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
- Storage: Cloudinary
